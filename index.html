Настройка Автоматизации GitHub

Укажи свой репозиторий, откуда сайт должен брать свежий файл data.json, собранный ИИ.

Как это работает?

1. Скрипт GitHub Actions (или Gemini Spark) каждый день собирает мету и кладет её в файл data.json в твоем репозитории dota-meta-tracker.

2. Этот дашборд скачивает её по прямой ссылке, обходя кэширование, и накладывает на твою личную статистику.
